PR #43: https://github.com/tucker2331-design/bill-tracker/pull/43.

Process upgrades proposed: Point 11 (*Side-Effect Gating Check*), dry-run with breaker artificially tripped, and monitor-as-bug-signal for counters that should be moving cycle-over-cycle but aren't (green-but-no-progress is a distinct failure mode).
